# @figlet-ts/lib

Library for rendering figlet / FLF fonts.

Use it for:

-   Displaying awesome ASCII fonts in your browser app
-   Displaying awesome ASCII fonts in your command-line tool

## Installation

```bash
# npm
npm install @figlet-ts/lib

# yarn
yarn add @figlet-ts/lib
```
